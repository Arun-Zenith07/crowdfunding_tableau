🚀 Kickstarter Crowdfunding Dashboard | Tableau Project

📊 Project Overview
This project presents an interactive **Kickstarter Crowdfunding Dashboard** built using **Tableau** to analyze crowdfunding campaign performance through KPI analysis and data visualization techniques.

The dashboard provides insights into:
- Funding trends
- Project success rates
- Backer engagement
- Category-wise performance
- Country-wise project distribution
- Goal range analysis

The objective of this project is to transform raw crowdfunding data into meaningful business insights that support data-driven decision-making.

---

🛠 Tools & Technologies Used
- Tableau
- Excel
- Data Visualization
- KPI Analysis
- Business Intelligence

---

📌 Key KPIs
- Total Categories
- Total Amount Raised
- Number of Backers
- Percentage of Successful Projects
- Average Number of Days for Successful Projects

---

📈 Dashboard Insights

🔹 Category Analysis
Analyzed project distribution across different categories to identify the most active crowdfunding segments.

🔹 Funding Performance
Compared categories based on the total amount raised to understand high-performing project domains.

🔹 Backer Analysis
Identified categories with the highest number of backers and audience engagement.

🔹 Success Rate Analysis
Evaluated project success percentages by:
- Year
- Month
- Category
- Goal Range

🔹 Country-wise Analysis
Explored project creation trends across different locations and regions.

🔹 Outcome Analysis
Visualized project outcomes to understand overall crowdfunding performance.

---

🎯 Key Business Insights
- Product Design projects generated the highest funding amounts.
- Tabletop Games attracted the highest number of backers.
- Overall project success rate reached 38.35%.
- Some categories achieved success rates above 70%.
- California recorded the highest number of projects.
